How to Create AWS ServerLess API
----------------------------------------------------
Step 1: Install the AWS Toolkit Extension in Visual studio
Step 2: Create AWS Account on AWS Portal
https://signin.aws.amazon.com/
and Configure the user Credensial on Visual Studio AWS ToolKit

Step 3: Create the Web Api Serverless Project
Step 4: Write the API Code in Controller folder
Step 5: Debug and test the code on local Enviorment
Step 6: Publish on AWs Portal using Visual Studio 2019.
